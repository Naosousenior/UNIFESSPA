A curso é formada por diferentes Instrutores, que é uma outra classe. Pelo menos um instrutor é necessário para ministrar o curso. Ela possui um método para descrever os dados do curso, outro para listar os Instrutores integrantes, e um outro para adicionar um novo instrutor ao curso. Por fim, temos um método simples para capturar o nome do curso.
A classe Instrutor representa os Instrutores do sistema. Ela tem um método ministrarCurso, que recebe como argumento um curso para o instrutor ministrar aula. Além disso, também possui um conjunto de métodos para obter os dados a respeito do instrutor.
